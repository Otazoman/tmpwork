【ソフト紹介】
本アプリケーションは、コンピュータ内の情報を収集・表示するソフトです。
PC 資産管理の情報取得ツールとして使用できます。

表示内容
    PC情報（一般）
        Windows バージョン、Windows ProductID
        IE バージョン、IE 更新バージョン、画面解像度
        物理メモリ、ページファイルの容量、使用量、空容量
        システムハードディスクの容量、使用量、空容量
        システムパス
        OEM マシンの場合、製造者、モデル名
        ドライブ情報(ドライブ、容量、空容量、ファイルシステム)
    CPU
        CPU一般名、ベンダーID、CPU の種類、動作クロック、キャッシュ、機能フラグ
        Type,Family,Model,Steppinng CPUシリアルナンバー
    ネットワーク（LAN 環境の場合）
        ユーザ名、コンピュータ名、ホスト名、ドメイン名、ワークグループ名
        コンピュータの説明、ネットワークプロバイダ名、IE設定環境
        IPアドレス、サブネットマスク、ゲートウェイ
        DHCPServer、PrimaryWINS、SecondaryWINS、
        PrimaryDNS、SecondaryDNS、アダプター名、MACアドレス
    アプリケーション
        インストールソフト一覧
    スタートアップ
        スタートアップソフト一覧
    プリンタ
        プリンタ一覧
    共有資源
        共有資源一覧
    環境変数
        環境変数一覧
    Device
        デバイス一覧
    WMI
        WMI(Windows Management Instrumentation) 情報
    保存
        上記、情報すべてをテキスト形式、又は CSV形式で保存します。
        自動保存・自動終了
        上記、情報をメールで送信できます。
        自動送信・自動終了
        環境ファイルの変更不可ができます。
＊注意
  自動終了を使用し、設定内容を変更したい場合は、
  PCView.exe と同じフォルダに PCView.ini ファイルが生成されます。
  この PCView.ini を削除し PCView を再度起動すれば、設定内容の変更ができます。
  または、補足説明 .txt に PCView.ini の設定方法がありますので、
  そちらを参考に直接編集していただいても結構です。
==============================================================================
【主な使用法】
　１．PCViewを起動後、保存ボタンで取得情報をテキストファイルに保存。
　２．LAN上のパソコンの共有フォルダにPCViewをコピーし、各パソコンから
	LAN越しにPCViewを実行し保存ボタンで、共有フォルダ内に、
	各パソコンの取得情報がテキストファイルに保存されます。
　３．メール送信ボタンで、指定相手先にメールで取得情報を送信します。
	ただし、SMTPサーバーアドレスが必要となります。
	また、Pop Before SMTP には、未対応。txt or csv をメールに添付できます。
　４．共有サーバーに置きログオンスクリプトにて自動起動・自動終了で情報収集
==============================================================================
【インストール】
　１．PCViewをダウンロードします。
　２．PCView0841.zipを解凍ツールを使用して任意のフォルダに解凍。
　３．あとは解凍先にあるPCView.exeを起動。
==============================================================================
【アンインストール】
  １．解凍したPCView関連ファイルを直接削除します。
  （レジストリには一切書き込みはありません）
==============================================================================
【ファイル】
	解凍後、フォルダには以下のファイルができます。
	PCView.exe	実行ファイル
	Readme .txt	今読んでいるファイル
	補足説明 .txt	INIファイルの説明
	PCView.ENU	英語環境用リソース
==============================================================================
【ログオンスクリプトの不具合回避方法】2003/11/28
  PCViewのｉｎｉがデフォルト状態になるという、情報をいただいておりましたが、
  PCView 側での対応が遅れておりますが、下記の方法で回避できるようです。

  下記のように各フォルダ毎にアクセス権を付けて、PCView.exeを
  ログオンスクリプトで起動すると、現象が出ません。

【アクセス権】（Win NT 4.0）
-------------------------------------------------------------
フォルダ名  ｜　アクセス権        ｜ ファイル名
-------------------------------------------------------------
PCView      ｜Everyone（読み取り）｜ PCView.exe　PCView.ini
-------------------------------------------------------------
PCView_Data ｜Everyone（変更）    ｜ ****.csv（データ）
-------------------------------------------------------------

【その他】
    ソフトのバージョン情報は、NT系のみで、9X系では表示されません。
    ネットワーク系の情報も Windows 95,NT で、一部表示できない場合があります。

　CSV形式
    Key        （保存ファイル名と同じ）
    Group      （グループ）
    Item       （大項目）
    SubItem    （小項目）
    Contents   （内容）
    Unit       （単位）
    Renewalday （更新日）

　メール送信時は、半角カタカナは全角カタカナに変更
　CSV出力時は、',' を '.' に変更
